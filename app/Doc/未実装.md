**現状:** `SettingsDataStore` に `scanSoundEnabledFlow` があり、`SettingsViewModel` でトグルも動くが、**実際に音を鳴らすコードが存在しない**。

2. `util/SoundHelper.kt` を新規作成:
```kotlin
object SoundHelper {
    private var soundPool: SoundPool? = null
    private var scanSoundId: Int = 0

    fun init(context: Context) {
        val attrs = AudioAttributes.Builder()
            .setUsage(AudioAttributes.USAGE_ASSISTANCE_SONIFICATION)
            .build()
        soundPool = SoundPool.Builder().setMaxStreams(1).setAudioAttributes(attrs).build()
        scanSoundId = soundPool!!.load(context, R.raw.scan_beep, 1)
    }

    fun playScanSound(enabled: Boolean) {
        if (enabled) soundPool?.play(scanSoundId, 1f, 1f, 0, 0, 1f)
    }

    fun release() { soundPool?.release(); soundPool = null }
}
```

3. `ScanViewModel.processBarcode()` 成功時に呼び出し
4. `OrderScanViewModel.processBarcode()` 成功時も同様
5. `MainActivity` の `onCreate`/`onDestroy` で `SoundHelper.init()`/`release()`

```kotlin
// getInjectPromptJsWithFallback() を新規追加
fun getInjectPromptJsWithFallback(selectors: List<String>, prompt: String): String {
    val selectorArray = selectors.joinToString(",") { "'$it'" }
    return """
        (function() {
            var selectors = [$selectorArray];
            var escaped = ${prompt.replace("'", "\\'")};
            for (var s of selectors) {
                var el = document.querySelector(s);
                if (el) {
                    el.focus();
                    el.textContent = escaped;
                    el.dispatchEvent(new Event('input', {bubbles: true}));
                    return true;
                }
            }
            return false;
        })()
    """.trimIndent()
}
```

### 🟢 8-3. フォーカスモードの輝度制御が Activity 依存
**現状:** `OrderListScreen.kt` で `context as Activity` とキャストしているが、Fragmentの再構成時にクラッシュする可能性がある。  
**修正:**
```kotlin
val activity = LocalContext.current as? ComponentActivity
DisposableEffect(isFocusMode) {
    if (!isFocusMode) return@DisposableEffect onDispose {}
    val window = activity?.window ?: return@DisposableEffect onDispose {}
    val originalBrightness = window.attributes.screenBrightness
    window.attributes = window.attributes.also {
        it.screenBrightness = WindowManager.LayoutParams.BRIGHTNESS_OVERRIDE_FULL
    }
    onDispose {
        window.attributes = window.attributes.also {
            it.screenBrightness = originalBrightness
        }
    }
}
```
