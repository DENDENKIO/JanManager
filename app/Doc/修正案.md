# JanManager アプリ 修正案

最終更新: 2026-04-05

---

## 概要

本ドキュメントはJanManagerアプリの以下の問題点に対する修正案をまとめたものです。

1. **包装単位の管理（最重要）** - 単品・パック・ケースの3種類に再設計し、削除・修正を可能にする
2. **包装紐づけモード（LinkageMode）の整理** - 現在リニューアル紐づけと包装紐づけが混在している問題を解消
3. **商品詳細画面（ProductDetailScreen）のUI改善** - 包装単位の登録・管理UIをスマートに刷新
4. **全般的なUI/UXの改善** - 見た目と使いやすさの向上

---

## 機能1: 包装単位の仕組み再設計（最重要）

### 現状の問題点

1. `ProductDetailScreen.kt` の包装単位セクションで、バーコードとPackageType(`CASE`固定)のみ入力でき、**単品・パック・ケースの選択ができない**
2. 追加した包装単位を**削除できない**（`PackageUnitDao`に`delete()`メソッドはあるが、UIに削除ボタンが存在しない）
3. `LinkageMode.kt` のスロットが「旧JAN・新JAN・パッケージ」となっており、仕様書の「単品・パック・ケース」と**完全にズレている**（リニューアル紐づけと包装紐づけが混在）
4. `ScanViewModel.kt`の`executeLinkage()`が`linkRenewal()`しか呼んでおらず、**包装単位の保存処理が未実装**

### 修正方針

#### 1-A. LinkageMode.kt を包装紐づけ専用に再設計

**修正対象:** `app/src/main/java/com/example/janmanager/ui/scan/components/LinkageMode.kt`

スロットを仕様書通りの「単品（PIECE）・パック（PACK）・ケース（CASE）」に変更し、スキャンしたJANコードをそれぞれのスロットに割り当てる。

```kotlin
// 修正後のLinkageMode.kt（抜粋）

@Composable
fun LinkageMode(viewModel: ScanViewModel) {
    val pieceJan by viewModel.linkagePieceJan.collectAsState()   // 単品
    val packJan  by viewModel.linkagePackJan.collectAsState()    // パック
    val caseJan  by viewModel.linkageCaseJan.collectAsState()    // ケース
    val packQty  by viewModel.linkagePackQty.collectAsState()    // パック入数
    val caseQty  by viewModel.linkageCaseQty.collectAsState()    // ケース入数
    val activeSlot by viewModel.activeLinkageSlot.collectAsState()

    Column(
        modifier = Modifier.fillMaxSize().padding(16.dp),
        verticalArrangement = Arrangement.spacedBy(12.dp)
    ) {
        Text("包装紐づけモード", style = MaterialTheme.typography.titleLarge)
        Text(
            "スキャンするスロットをタップして選択し、バーコードを読み取ってください。",
            style = MaterialTheme.typography.bodySmall,
            color = MaterialTheme.colorScheme.onSurfaceVariant
        )

        // 単品スロット
        PackageSlotCard(
            label = "単品",
            icon = Icons.Default.CropFree,
            value = pieceJan,
            isActive = activeSlot == LinkageSlot.PIECE,
            onClick = { viewModel.setLinkageSlot(LinkageSlot.PIECE) },
            onClear = { viewModel.clearLinkageSlot(LinkageSlot.PIECE) }
        )

        // パックスロット（入数入力付き）
        PackageSlotCard(
            label = "パック",
            icon = Icons.Default.ViewModule,
            value = packJan,
            isActive = activeSlot == LinkageSlot.PACK,
            onClick = { viewModel.setLinkageSlot(LinkageSlot.PACK) },
            onClear = { viewModel.clearLinkageSlot(LinkageSlot.PACK) },
            quantity = packQty,
            onQuantityChange = { viewModel.setPackQty(it) }
        )

        // ケーススロット（入数入力付き）
        PackageSlotCard(
            label = "ケース",
            icon = Icons.Default.Inventory2,
            value = caseJan,
            isActive = activeSlot == LinkageSlot.CASE,
            onClick = { viewModel.setLinkageSlot(LinkageSlot.CASE) },
            onClear = { viewModel.clearLinkageSlot(LinkageSlot.CASE) },
            quantity = caseQty,
            onQuantityChange = { viewModel.setCaseQty(it) }
        )

        // 対象商品の単品JANを先に確認する仕組み（バリデーション）
        val canRegister = pieceJan.isNotEmpty() && (packJan.isNotEmpty() || caseJan.isNotEmpty())
        Button(
            onClick = { viewModel.executePackageLinkage() },
            modifier = Modifier.fillMaxWidth(),
            enabled = canRegister
        ) {
            Text("包装単位を登録する")
        }

        if (!canRegister) {
            Text(
                "※ 単品JANと、パックまたはケースのJANが必要です",
                style = MaterialTheme.typography.bodySmall,
                color = MaterialTheme.colorScheme.error
            )
        }
    }
}
```

#### 1-C. ScanViewModel.kt の包装紐づけ State を追加

**修正対象:** `ScanViewModel.kt`

```kotlin
// 既存のリニューアル紐づけ用変数は削除し、包装紐づけ用に置き換える

// 包装紐づけ State
private val _linkagePieceJan = MutableStateFlow("")
val linkagePieceJan = _linkagePieceJan.asStateFlow()

private val _linkagePackJan = MutableStateFlow("")
val linkagePackJan = _linkagePackJan.asStateFlow()

private val _linkageCaseJan = MutableStateFlow("")
val linkageCaseJan = _linkageCaseJan.asStateFlow()

private val _linkagePackQty = MutableStateFlow(6)   // デフォルト6入り
val linkagePackQty = _linkagePackQty.asStateFlow()

private val _linkageCaseQty = MutableStateFlow(12)  // デフォルト12入り
val linkageCaseQty = _linkageCaseQty.asStateFlow()

fun clearLinkageSlot(slot: LinkageSlot) {
    when (slot) {
        LinkageSlot.PIECE -> _linkagePieceJan.value = ""
        LinkageSlot.PACK  -> _linkagePackJan.value  = ""
        LinkageSlot.CASE  -> _linkageCaseJan.value  = ""
    }
}

fun setPackQty(qty: Int) { _linkagePackQty.value = qty }
fun setCaseQty(qty: Int) { _linkageCaseQty.value = qty }

// processBarcode() 内の LINKAGE 分岐を修正
ScanModeTab.LINKAGE -> {
    when (_activeLinkageSlot.value) {
        LinkageSlot.PIECE -> {
            _linkagePieceJan.value = normalizedBarcode
            // 単品スキャン後は自動的にパックへ移動
            _activeLinkageSlot.value = LinkageSlot.PACK
        }
        LinkageSlot.PACK -> {
            _linkagePackJan.value = normalizedBarcode
            _activeLinkageSlot.value = LinkageSlot.CASE
        }
        LinkageSlot.CASE -> {
            _linkageCaseJan.value = normalizedBarcode
            // ケースまで完了、そのまま待機
        }
    }
}

// 新しい executePackageLinkage()
fun executePackageLinkage() {
    viewModelScope.launch {
        val pieceJan = _linkagePieceJan.value
        if (pieceJan.isEmpty()) return@launch

        val product = productRepository.getProductByJan(pieceJan) ?: return@launch

        // パック包装単位の保存
        if (_linkagePackJan.value.isNotEmpty()) {
            val packUnit = PackageUnit(
                productId = product.id,
                barcode = _linkagePackJan.value,
                barcodeType = JanCodeUtil.detectCodeType(_linkagePackJan.value),
                packageType = PackageType.PACK,
                packageLabel = "パック",
                quantityPerUnit = _linkagePackQty.value
            )
            productRepository.addPackageUnit(packUnit)
        }

        // ケース包装単位の保存
        if (_linkageCaseJan.value.isNotEmpty()) {
            val caseUnit = PackageUnit(
                productId = product.id,
                barcode = _linkageCaseJan.value,
                barcodeType = JanCodeUtil.detectCodeType(_linkageCaseJan.value),
                packageType = PackageType.CASE,
                packageLabel = "ケース",
                quantityPerUnit = _linkageCaseQty.value
            )
            productRepository.addPackageUnit(caseUnit)
        }

        // リセット
        _linkagePieceJan.value  = ""
        _linkagePackJan.value   = ""
        _linkageCaseJan.value   = ""
        _activeLinkageSlot.value = LinkageSlot.PIECE
    }
}
```

@Composable
fun PackageUnitRow(pack: PackageUnit, onDelete: () -> Unit) {
    val typeLabel = when (pack.packageType) {
        PackageType.PIECE -> "単品"
        PackageType.PACK  -> "パック"
        PackageType.CASE  -> "ケース"
    }
    val icon = when (pack.packageType) {
        PackageType.PIECE -> Icons.Default.CropFree
        PackageType.PACK  -> Icons.Default.ViewModule
        PackageType.CASE  -> Icons.Default.Inventory2
    }

    Card(modifier = Modifier.fillMaxWidth()) {
        Row(
            modifier = Modifier.padding(horizontal = 12.dp, vertical = 8.dp),
            verticalAlignment = Alignment.CenterVertically
        ) {
            Icon(icon, contentDescription = typeLabel, modifier = Modifier.size(24.dp))
            Spacer(Modifier.width(8.dp))
            Column(modifier = Modifier.weight(1f)) {
                Text(typeLabel, style = MaterialTheme.typography.labelMedium)
                Text(pack.barcode, style = MaterialTheme.typography.bodyMedium)
                if (pack.packageType != PackageType.PIECE) {
                    Text("${pack.quantityPerUnit}入", style = MaterialTheme.typography.bodySmall,
                         color = MaterialTheme.colorScheme.onSurfaceVariant)
                }
            }
            // 削除ボタン（確認ダイアログ付き）
            var showConfirm by remember { mutableStateOf(false) }
            IconButton(onClick = { showConfirm = true }) {
                Icon(Icons.Default.DeleteOutline, contentDescription = "削除",
                     tint = MaterialTheme.colorScheme.error)
            }
            if (showConfirm) {
                AlertDialog(
                    onDismissRequest = { showConfirm = false },
                    title   = { Text("包装単位を削除") },
                    text    = { Text("${typeLabel}（${pack.barcode}）を削除しますか？") },
                    confirmButton = {
                        TextButton(onClick = { onDelete(); showConfirm = false }) { Text("削除") }
                    },
                    dismissButton = {
                        TextButton(onClick = { showConfirm = false }) { Text("キャンセル") }
                    }
                )
            }
        }
    }
}

## 機能3: 包装紐づけモードとリニューアル紐づけの分離

### 現状の問題点

`ScanViewModel.kt` と `LinkageMode.kt` では現在、スロット名が「旧JAN・新JAN・パッケージ」となっており、リニューアル管理と包装単位管理が1つのモードに混在している。仕様書では包装紐づけモードは「単品・パック・ケース」専用とされている。

### 修正方針

- **包装紐づけモード（LinkageMode）**: 単品・パック・ケースの紐づけのみを担当（機能1で対応）
- **リニューアル紐づけ**: `ProductDetailScreen.kt` の「リニューアル・終売」セクションに既存の入力フォームがあるため、そちらで完結させる

リニューアル紐づけをスキャンモードから行いたい場合は、将来的に「確認モード」からリニューアル設定ダイアログを開く方式が推奨。

@Composable
fun PackageSlotCard(
    label: String,
    icon: ImageVector,
    value: String,
    isActive: Boolean,
    onClick: () -> Unit,
    onClear: () -> Unit,
    quantity: Int? = null,
    onQuantityChange: ((Int) -> Unit)? = null
) {
    Card(
        modifier = Modifier.fillMaxWidth().clickable(onClick = onClick),
        border   = if (isActive) BorderStroke(2.dp, MaterialTheme.colorScheme.primary) else null,
        colors   = CardDefaults.cardColors(
            containerColor = if (isActive)
                MaterialTheme.colorScheme.primaryContainer
            else
                MaterialTheme.colorScheme.surfaceVariant
        )
    ) {
        Row(
            modifier = Modifier.padding(12.dp),
            verticalAlignment = Alignment.CenterVertically
        ) {
            Icon(icon, contentDescription = label, modifier = Modifier.size(28.dp))
            Spacer(Modifier.width(12.dp))
            Column(modifier = Modifier.weight(1f)) {
                Text(label, style = MaterialTheme.typography.labelMedium)
                Text(
                    text  = value.ifEmpty { if (isActive) "▶ スキャン待ち" else "未スキャン" },
                    style = MaterialTheme.typography.bodyLarge,
                    color = if (value.isEmpty()) MaterialTheme.colorScheme.onSurfaceVariant
                            else MaterialTheme.colorScheme.onSurface
                )
                // 入数入力欄（パック・ケースのみ）
                if (quantity != null && onQuantityChange != null && value.isNotEmpty()) {
                    OutlinedTextField(
                        value         = quantity.toString(),
                        onValueChange = { it.toIntOrNull()?.let(onQuantityChange) },
                        label         = { Text("入数") },
                        keyboardOptions = KeyboardOptions(keyboardType = KeyboardType.Number),
                        modifier      = Modifier.fillMaxWidth().padding(top = 4.dp),
                        singleLine    = true
                    )
                }
            }
            // クリアボタン（スキャン済みの場合のみ表示）
            if (value.isNotEmpty()) {
                IconButton(onClick = onClear) {
                    Icon(Icons.Default.Close, contentDescription = "クリア",
                         tint = MaterialTheme.colorScheme.onSurfaceVariant)
                }
            }
        }
    }
}
